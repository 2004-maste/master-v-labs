MASTER-V LABS WEBSITE - DEPLOYMENT INSTRUCTIONS

1. FOLDER STRUCTURE:
   Create these files in a folder called "master-v-labs":
   - index.html
   - style.css
   - script.js
   - images/
        - logo.png (your logo)
        - background.png (your background image)

2. ADD YOUR IMAGES:
   - Save your logo as "logo.png" in the "images" folder
   - Save your background as "background.png" in the "images" folder
   - Recommended logo size: at least 200x200 pixels
   - Recommended background size: 1920x1080 pixels or larger
   - Formats: PNG with transparent background for logo, JPG/PNG for background

3. TEST LOCALLY:
   - Open index.html in your web browser
   - Check that the background image appears on all sections
   - Verify all text is readable against the background

4. GET ONLINE:
   Option A: Netlify (Free & Easy)
   - Go to netlify.com
   - Drag your entire "master-v-labs" folder to their upload area
   - Your site will be live instantly!

   Option B: Your own hosting
   - Upload all files to your web hosting via FTP/cPanel
   - Point your domain mastervlabs.dev to this hosting

5. BACKGROUND IMAGE TIPS:
   - Use a high-quality image (1920x1080 or larger)
   - Choose an image that's not too busy so text remains readable
   - The code includes white overlays to ensure text readability
   - You can change the background image anytime by replacing background.png

6. CUSTOMIZE LATER:
   - To adjust background overlay: Change the rgba values in style.css
   - To change colors: Edit the CSS color variables
   - To update services: Edit the services-grid in index.html

The background image will now appear across all sections with proper overlays for readability!
